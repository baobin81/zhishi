## 国际站

# 开始
1. 安装 

    安装参考: https://laravel-china.org/docs/laravel/5.7/installation/2242

2. 新建一个项目

    `laravel new blog`

3.  本地开发环境

    `php artisan serve`

# 问题
1. 如何查当前版本？
   
    在项目根目录 cmd 执行 `php artisan -V`
2. 获得artisan的帮助
    
     cmd 执行 ` php artisan help `
